Gospelgo! Desktop

Early access. Make suggestions to influence how the game progresses. Play from desktop browser.

Desktop Version

THis will be a desktop complementary version of the mobile Gospelgo! game. The players progress will overlap between both versions although some features may be different.

Content

Gospelgo is being developed kind of in reverse. The game is live but without content. The content (items, quests, characters, storylines, etc.) will be determined and added over time.

Inventory/Database

Inventory and database systems will likely be developed as the game grows.

Web based

Gospelgo is being developed to run in a web browser. No downloads or installs required. Just visit a web address and play. Login with email and password and access cloud features for all Gospelgo games. A refresh token will automatically keep you logged in for a period of time.

About Gospelgo

Gospelgo is an indie game project that creates free web-games aimed at helping players understand, hear and live the gospel of Jesus Christ. Because the games run in a browser, anyone can play them instantly from anywhere and on most any device (some are exclusively mobile or desktop).

Gosplego uses a continous-deployment pipeline for quick turnaround time of updates and bug fixes.

Go map game

Gospelgo, a go map game is in the very early stages of development. So Gospelgo is the name of the project and the name of two planned games, Gospelgo and Gosplego Desktop. Other games are and will be available as well such as Word, Big Heart and Gospelgo Bubble Shooter.

Easy access

Gospelgo games are easy to access by visiting a simple web address.Ths will bypass gate keepers like some game platforms, but those platforms have large audiences so it is a trade-off. Pray that those who need to find Gospelg games will do so. God will work it out. He can do anything!

Free games

The plan is that Gospelgo core games will remain free. There are costs associated with bandwidth usage, web calls, cloud storage and the like, so donation links will be provided for those who want to give/support, but there is no pressure to do so. Also, Gospelgo may release some games that are designed to bring in money, for those who prefer to support the project by playing a game.

Tech

Gospelgo uses a tech stack that allows for quick deployment of changes, updates and bug fixes. This includes source control, scalable hosting and webhooks for automatic publishing of games, the website and discord notifications. The discord posts include deploy messages which serve as dev notes.
